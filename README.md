# JAGGAER Elevator Review Pack

This folder contains the consolidated proposal in two parts:

- `content.csv`: the Excel-friendly content sheet with all copy for the site
- `index.html`: the single-page HTML proposal that reads from `content.csv`

## How to review and edit the wording

1. Open `content.csv` in Excel.
2. Review and edit the wording row by row.
3. Save it again as CSV.
4. Keep the same column names in the first row.

## Media slots

Every page already includes a media row.

Use these columns:

- `media_type`: `image`, `video`, or `embed`
- `media_url`: the URL you want to display on that page
- `media_caption`: optional caption text

## Notes

- The proposal is consolidated under one name: `JAGGAER Elevator`.
- The three capabilities are presented as modules within the same proposal.
- If the browser blocks local CSV loading, use the file picker in `index.html` to load the CSV manually after exporting it from Excel.
